- [MIT 6.S191 Introduction to Deep Learning](http://introtodeeplearning.com/)

- [TensorFlow.js Deep Learning with JavaScript by Deeplizard](https://www.youtube.com/playlist?list=PLZbbT5o_s2xr83l8w44N_g3pygvajLrJ-)
